# **App Name**: Cosmic Insights

## Core Features:

- Cálculo do Mapa Astral: Calcular o mapa astral com base nos dados de nascimento fornecidos (data, hora, local).
- Posições Planetárias: Exibir as posições do Sol, Lua, Ascendente e outros planetas nas casas astrológicas.
- Interpretação do Mapa: Fornecer interpretações do mapa, incluindo traços de personalidade, aspectos do caminho de vida e potenciais.
- Análise de Trânsitos: Analisar os trânsitos planetários atuais em relação ao mapa astral. Esta ferramenta de IA ajuda a identificar fases, ciclos, oportunidades e desafios importantes.
- Geração de Mapa Personalizado: Gerar uma representação visual do mapa astral com base nos dados de nascimento do usuário.

## Style Guidelines:

- Cor primária: Indigo Profundo (#4B0082) para evocar uma sensação de mistério e energia cósmica.
- Cor de fundo: Cinza Escuro (#222222) para fornecer contraste e melhorar a visibilidade dos elementos celestiais.
- Cor de destaque: Roxo Elétrico (#BF00FF) para destacar elementos interativos e aspectos astrológicos importantes.
- Fonte do texto do corpo: 'Literata', serifada, para uma sensação de leitura literária apropriada à explicação astrológica.
- Fonte do título: 'Space Grotesk', sans-serif, para transmitir o tema cósmico de uma forma que pareça apropriadamente futurista.
- Usar ícones elegantes e minimalistas representando os planetas, signos do zodíaco e casas astrológicas.
- Manter um layout limpo e estruturado, com foco na clareza e facilidade de navegação pelos dados astrológicos.
- Implementar animações sutis ao exibir posições planetárias e interpretações de mapa para adicionar dinamismo.
